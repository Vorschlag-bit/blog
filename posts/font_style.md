## 글꼴 관련 스타일(font)
텍스트 스타일에는 글자의 모양새를 지정하는 <b>글꼴 스타일(font)</b>과 웹 문서에 표시되는 텍스트를 지정하는 
<b>문단 스타일</b>이 있다. <b>'font'</b>로 시작하는 속성은 글꼴과 관련이 있다.

### font-family 속성
웹 문서에서 사용하는 글꼴은 <code>font-family</code> 속성으로 지정한다. 이 속성은 `<body` 태그를 비롯해
`<p>` 태그나 `<h1>` 태그처럼 텍스트를 사용하는 요소에 사용한다.

```CSS
p { font-family: 글꼴 이름 }
```
웹 문서의 텍스트는 사용자 시스템의 글꼴을 이용 웹 브라우저 화면에 표시된다. 웹 문서에 지정한 글꼴이 사용자 시스템에 설치되어 있지 않다면 웹 문서에서
지정한 글꼴이 아닌 글꼴이 보이게 된다. 따라서 웹 문서에서 글꼴을 지정할 때는 한 가지 글꼴만 선택하기도 하지만 글꼴이 없을 경우를 대비해서 2,3번째 글꼴도
지정한다.
```CSS
p { font-family: '맑은 고딕', 돋움, 굴림 }
```
---

### font-size 속성
글자 크기를 따로 지정하지 않더라도 제목 테스트는 본문 텍스트보다 굵고 크게 표시된다. 이는 웹 브라우저마다 기본 스타일이 적용되어 있기 때문이다.  
글자 크기를 원하는 크기로 조절하고 싶다면 `font-size` 속성을 사용하면 된다. 글자 크기는 <b>절대 크기</b> 혹은 <b>상대 크기</b>로 지정할 수 있다.

```CSS
p { font-size: 40px }
```

#### 절대 크기로 글자 크기 지정
절대 크기는 글자 크기를 고정하는 방법으로 주로 <b>pixel 단위</b>를 사용한다.  
또한 pixel말고도 아래와 같은 <b>키워드</b>를 사용해 지정할 수도 있다.

```CSS
p { font-size: xx-small || x-small || small || medium || large || x-large || xx-large }
```

#### 상대 크기로 글자 크기 지정
글자 크기를 상대적으로 지정한다는 건 기준값이 존재하고 그 값의 몇 배인지릂 표시하는 방법이며 <b>%</b>나 <b>em,rem</b> 단위를 사용한다.  

%나 em은 <b>부모 요소의 글씨 크기를 기준으로</b> 상대적인 값을 지정한다. 따라서 부모 요소의 글자 크기가 달라지면 %나 em을 사용한 요소의 글자 크기도
달라진다. 

rem은 웹 문서의 기본 글자 크기를 미리 정하고, 원하는 요소의 글자 크기를 상댓값으로 지정한다. 예를 들어 문서의 기본 글자 크기가 16px이고, 제목의
글자 크기를 2rem으로 정하면 제목은 32px(16 * 2)이 된다.

### font-style 속성
글자를 이탤릭체로 표시할 때는 `font-style` 속성을 사용한다. 이탤릭체로 바꾸는 속성값은 `italic`과 `oblique`가 있고, 주로
`italic` 속성을 사용한다.

```CSS
p { font-style: normal || italic || oblique }
```
---

### font-weight 속성
글자 굵기를 지정하는 속성이다. 미리 존재하는 예약어(<b>normal,bold,bolder</b>)나 숫자 값을 통해 굵기를 지정할 수 있다.

100~900 사이로 지정이 가능하고 400은 normal, 700은 bold에 해당한다. 당연히 예약어 대신 숫자 값을 사용하면 굵기를 더 세밀하게 조절할 수 있다.

- **`font-family`**: 텍스트의 글꼴을 지정하는 속성. 사용자 시스템에 해당 글꼴이 설치되어 있지 않을 경우를 대비해 **여러 개의 예비 글꼴**을 쉼표로 구분하여 순서대로 지정해야 한다.
- **`font-size`**: 글자의 크기를 조절하는 속성. `px`나 키워드를 사용하는 **절대 크기** 방식과, 부모 요소(`%`, `em`)나 최상위 요소(`rem`)를 기준으로 하는 **상대 크기** 방식이 있다.
- **`font-style`**: 글자를 기울여서 표시할 때 사용하며, 주로 `italic` 속성값을 사용하여 이탤릭체를 적용한다.
- **`font-weight`**: 글자의 굵기를 지정하는 속성. `bold`와 같은 예약어뿐만 아니라 **100~900 사이의 숫자 값**을 사용하여 굵기를 세밀하게 조절할 수 있다.
