## 스타일과 스타일 시트
웹 문서에서 <b>스타일</b>이란 HTML 문서에서 자주 사용되는 글꼴,색상과 같이 문서의 겉모습을 결정하는 걸 말한다.  
웹 문서의 <b>내용과는 상관없이 순수하게 디자인을 손보는 것</b>이다.

기존 HTML 문서는 PC의 웹 브라우저 화면을 기본으로 하기 때문에 인쇄하려면 용지에 맞게 새로 작성해야 했다.  
뿐만 아니라 모바일 홈페이지가 필요할 경우에도 스마트폰 브라우저에 맞게 따로 문서를 만들어야 했다.

하지만 HTML로 작성한 내용은 그대로 두고 대상 기기에 맞게 CSS만 따로 적용해 같은 내용을 서로 다른 기기에서 볼 수 있는
<b>반응형 웹 디자인</b>이 대두되면서 스타일의 역할이 더더욱 중요해졌다.

### 스타일 형식
CSS 스타일은 아래와 같다.
```CSS
선택자 { 속성1: 속성1값; 속성2: 속성2값; }
```
맨 앞에는 스타일을 적용할 선택자를 작성하고 중괄호를 통해서 선택자에 변경할 속성과 속성값을 지정하면 된다.
```HTML
span { text-align: center; color: red }
```

### 스타일 시트
웹 문서 내부에서는 여러 개의 스타일 규칙이 존재한다. 이런 스타일 규칙을 한눈에 확인하고 필요할 때마다 수정하기 쉽도록
한군데에 묶어 놓은 걸 <b>스타일 시트</b>라고 한다. 스타일 시트는 크게 웹 브라우저를 기본으로 하는 <b>브라우저 기본 스타일</b>과
개발자가 만드는 <b>제작자 스타일</b>로 나눌 수 있다. 제작자 스타일은 세부적으로 <b>인라인 스타일</b>과 <b>내부 스타일</b>,
<b>외부 스타일</b>로 나눌 수 있다.

#### 브라우저 기본 스타일
CSS를 사용하지 않은 웹 문서일지라도 브라우저에 표시될 때 자동으로 스타일이 적용된다. 이를 <b>브라우저 기본 스타일</b>이라고 부른다.
`<h1>` 태그를 사용하면 가장 큰 제목을 사용하고, `<span>` 태그를 사용하면 본문 글씨처럼 작은 글씨가 나오는 예시가 그런 것이다.
```HTML
<h1>제일 큰 제목</h1>
<span>이어서 붙는 성질의 글</span>
```

#### 인라인 스타일
간단한 스타일 정보라면 스타일 시트를 사용하지 않고 스타일을 적용할 대상에 직접 표시한다.  
이러한 방법을 <b>인라인 스타일</b>이라고 한다. 스타일을 적용하고 싶은 부분이 있다면 해당 태그에 `style` 속성을 통해
`style="속성: 속성값";` 형태로 스타일을 지정할 수 있다.

### 내부 스타일 시트
웹 문서 안에서 사용할 스타일을 같은 문서 안에 정리한 걸 <b>내부 스타일 시트</b>라고 한다.  
```HTML
<style>
    h1 {
        text-align: center;
        color: '#fff';
    }
</style>
```

### 외부 스타일 시트
웹 사이트를 만들 때 여러 웹 문서를 준비하게 된다. 그때마다 웹 문서마다 같은 내부 스타일 시트를 만들면
서버 공간은 물론 문서를 내려받는 시간까지 낭비가 된다.

따라서 웹 사이트를 제작 시에는 여러 웹 문서에서 사용할 스타일을 별도 파일로 저장하고 필요할 때마다 가져와서 사용한다.  
이렇게 따로 저장한 스타일 정보를 <b>외부 스타일 시트</b>라고 부르며 <b>*.css</b>라는 파일 확장자를 사용한다.
대표적으로 `global.css`가 있다.

외부 스타일 시트 파일을 작성 시에는 `<style>` 태그를 작성하지 않는다. 그리고 이렇게 만든 외부 스타일 시트는
웹 문서에 연결해야 적용이 된다. 외부 스타일 시트를 연결할 때는 `<link>` 태그를 사용한다.

```HTML
<link rel="stylesheet" href="외부 스타일 시트 파일 경로">
```

- **스타일(Style)의 정의와 중요성**
    - 웹 문서의 내용(HTML)과 별개로 디자인(겉모습)을 결정하는 요소.
    - 기기에 따라 디자인을 달리하는 **반응형 웹 디자인** 구현을 위해 필수적임.

- **CSS 기본 형식**
    - `선택자 { 속성: 속성값; }`의 형태를 가짐. (예: `span { color: red; }`)

- **스타일 시트(Style Sheet)의 분류**
    - **브라우저 기본 스타일**: CSS 없이도 브라우저가 태그에 자동으로 적용하는 기본 서식.
    - **제작자 스타일** (개발자가 작성):
        - **인라인 스타일(Inline)**: 태그 내에 `style` 속성으로 직접 적용.
        - **내부 스타일 시트(Internal)**: HTML 문서 내 `<style>` 태그 안에 모아서 작성.
        - **외부 스타일 시트(External)**: 별도의 `*.css` 파일로 저장 후 `<link>` 태그로 연결하며, 서버 효율과 관리 측면에서 가장 권장됨.
