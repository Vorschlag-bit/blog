## 스타일과 스타일 시트
웹 문서에서 <b>스타일</b>이란 HTML 문서에서 자주 사용되는 글꼴,색상과 같이 문서의 겉모습을 결정하는 걸 말한다.  
웹 문서의 <b>내용과는 상관없이 순수하게 디자인을 손보는 것</b>이다.

기존 HTML 문서는 PC의 웹 브라우저 화면을 기본으로 하기 때문에 인쇄하려면 용지에 맞게 새로 작성해야 했다.  
뿐만 아니라 모바일 홈페이지가 필요할 경우에도 스마트폰 브라우저에 맞게 따로 문서를 만들어야 했다.

하지만 HTML로 작성한 내용은 그대로 두고 대상 기기에 맞게 CSS만 따로 적용해 같은 내용을 서로 다른 기기에서 볼 수 있는
<b>반응형 웹 디자인</b>이 대두되면서 스타일의 역할이 더더욱 중요해졌다.

### 스타일 형식
CSS 스타일은 아래와 같다.
```CSS
선택자 { 속성1: 속성1값; 속성2: 속성2값; }
```
맨 앞에는 스타일을 적용할 선택자를 작성하고 중괄호를 통해서 선택자에 변경할 속성과 속성값을 지정하면 된다.
```HTML
span { text-align: center; color: red }
```

### 스타일 시트
웹 문서 내부에서는 여러 개의 스타일 규칙이 존재한다. 이런 스타일 규칙을 한눈에 확인하고 필요할 때마다 수정하기 쉽도록
한군데에 묶어 놓은 걸 <b>스타일 시트</b>라고 한다. 스타일 시트는 크게 웹 브라우저를 기본으로 하는 <b>브라우저 기본 스타일</b>과
개발자가 만드는 <b>제작자 스타일</b>로 나눌 수 있다. 제작자 스타일은 세부적으로 <b>인라인 스타일</b>과 <b>내부 스타일</b>,
<b>외부 스타일</b>로 나눌 수 있다.
