## 상댓값으로 글자 크기 지정하기 (em,rem)
글자 크기를 지정할 때는 pixel 단위를 많이 사용한다. 하지만 글자 크기를 pixel로 지정하면 <b>글자 크기가 고정</b>된다. 
반응형 웹 디자인을 목표로 제작된 사이트에선 화면 크기에 다라 글자 크기 역시 달리지게 만들어야 하므로 px 단위는
적합하지 않다. 이럴 때는 `em`이나 `rem` 같은 상대적인 글자 크기 단위를 사용하면 된다.

### em
`em`은 <b>부모 요소에서 지정한 글꼴의 대문자 M의 너비를 1em으로 놓고 상대적으로 값을 계산</b>한다.
부모 요소에서 글자 크기를 지정하지 않았다면 `body` 요소의 기본 크기를 사용한다.

```HTML
<style>
    p { font-size: 1em; }
    .content { font-size: 1.5em; }
</style>

<h1>제목</h1>
<p>p 태그 내용 1</p>
<div class="content">
    <p>p 태그 내용2</p>
    <p>p 태그 내용3</p>
</div>
```
위와 같은 구조가 있을 경우 1번째 요소는 16px(1em) 2,3번째 p 태그 내용은 24px(1.5em * 16px)으로 계산된다. 부모 태그인 `<h1>` 태그의 값을 기본으로 em이
계산되었기 때문이다. 이렇듯 웹 문서에서 요소 포함 관계가 복잡하거나 중간에 글자 크기가 바뀌는 등 예상치 못한 결과로 인해 `em`으로 설정한 글자의 크기가
이상하게 나올 수 있다.

### rem
em에서 부모 요소의 글자 크기 때문에 예상치 못한 결과가 나오는 걸 막기 위해 새로 생긴 단위가 `rem`이다. `rem`은 root에서 지정한 크기를 기준값으로 사용한다.
따라서 중간에 값이 바뀔 염려가 사라진다.

```HTML
<style>
    html { font-size: 16px; }
    p { font-size: 1.5rem }
</style>

<p>p 태그 내용</p>
```

위와 같은 스타일이 주어지면 1.5rem은 html 선택자에서 지정한 크기인 16px을 기준으로 하므로 24px의 값을 부모 요소에 상관없이 갖게 된다.

### 반응형 이미지
반응형 이미지란 <b>부모 요소의 너비에 따라 높이와 너비가 달라지는 이미지</b>를 말한다. 이미지는 웹 문서에 삽입할 때부터 크기가 고정되어 있으므로 브라우저 화면이
작아지면 이미지의 일부만 보이게 된다.

#### width: 100%
이미지의 너비를 100%로 지정하면 이미지가 부모 요소의 너비에 맞춰 항상 100%가 된다. 부모 요소의 너비가 너무 클 경우 이미지가 원본보다 커져서(pixel 이미지인 경우)
화질이 떨어질 수 있다.

#### max-width: 100%
`width` 속성과 함께 `max-width` 속성을 사용하면 부모 요소의 크기에 따라 이미지가 커질 때 원본 이미지의 크기만큼 커지게 된다. 최대 크기가 고정되니 화질이
떨어질 일이 없어서 다양한 크기의 화면에서도 이미지가 깔끔하게 보인다. 그래서 보통 반응형 이미지를 만들 때는 `width`와 `max-width` 속성을 함께 사용한다.

```CSS
img {
    max-width: 100%;
    width: 100%
}
```

### object-fit
웹 문서에서 이미지나 비디오 같은 시각 요소를 반응형으로 만들어서 크기가 바뀌면 원래 이미지나 비디오의 가로-세로 비율을 그대로 유지하는 게 쉽지 않다.
이럴 때는 `object-fit` 속성을 사용해 콘텐츠(이미지나 비디오)의 가로세로 비율을 유지하면서 해상도에 맞게 크기를 조절할 수 있다.

`object-fit`에서 사용할 수 있는 속성값을 표로 정리했다.

### 반응형 레이아웃
사이트가 반응형으로 동작하려면 레이아웃이 유연해야 한다. 넓은 화면일 때는 열을 여러 개 배치하고 좁은 화면일 때는 1개씩 배치하는 것처럼 레이아웃을
자유롭게 조절할 수 있어야 한다. 이런 레이아웃을 만들기 위해선 최근에 등장한 `flex` 박스 레이아웃과 CSS 그리드 레이아웃을 사용해야 한다.

반응형 웹을 만들기 위해선 미디어 쿼리에 대해서도 알아야 한다. 미디어 쿼리는 사이트에 접속하는 기기의 화면 가로 너비에 따라 레이아웃을 다르게 표시한다.

- <b>반응형 웹 글자 크기(상대 단위)</b>: `em`은 부모 요소 기준으로 계산되어 중첩에 따라 값이 달라질 수 있고, `rem`은 루트(html) 기준이라 예측 가능하고 일관된 크기 제어에 유리함.  
- **반응형 이미지**: 보통 `width: 100%`와 `max-width: 100%`를 함께 써서 부모 너비에 맞춰 줄어들되 원본보다 커지지 않게 하고 화질 저하를 방지함.  
- **object-fit**: 이미지나 비디오의 가로세로 비율을 유지하면서 요소 영역에 맞추려면 `object-fit`을 사용함. 주요 값은 `fill`, `contain`, `cover`, `none`, `scale-down`임.  
- **반응형 레이아웃**: 레이아웃은 유연해야 함. `flexbox`와 `CSS Grid`로 열 배치와 정렬을 조절하고, `@media` 쿼리로 화면 너비에 따라 레이아웃을 전환하면 다양한 기기에서 일관된 UX를 제공할 수 있음.
