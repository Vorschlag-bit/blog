## 기본 선택자
스타일 규칙은 태그뿐만 아니라 웹 문서의 다양한 <b>요소(element)</b>에도 적용이 가능하다.  
<b>선택자</b>는 웹 문서에서 어느 부분에 스타일을 적용할지 알려주는 것이다. 

이 글은 이 중에서 자주 사용되는 기본 선택자들을 알아보는 글.

### 1. 전체 선택자
<b>전체 선택자</b>는 말 그대로 문서의 모든 요소에 스타일을 적용할 때 사용한다. 주로 글꼴과 같이 모든 문서의 내부 요소에
공통적으로 적용하는 요소들에 사용한다. 사용법은 <b>*</b>를 붙이면 된다.

```CSS
* { attr: value, ... }
```

### 2. 타입 선택자
<b>타입 선택자</b>는 특정 태그를 사용한 모든 요소에 스타일을 적용한다. 즉, 타입 선택자를 사용해 스타일 지정 시
해당 태그를 사용하는 모든 요소에 적용된다는 의미이다.

```CSS
span { text-align: center; }
```

위 코드는 `<span>` 선택자를 사용하여 웹 문서에 있는 모든 `<span>` 태그를 쓴 요소에 모두 적용이 된다.

### 3. 클래스 선택자
특정 태그의 특정 부분에만 스타일을 적용하고 싶을 때는 <b>클래스 선택자</b>를 사용한다.  
클래스 선택자는 <b>클래스 이름(class)</b>을 사용해서 다른 선택자와 구분하는데, 이때 클래스 이름 앞에 <b>마침표(.)</b>를 반드시
찍어야 한다.

```CSS
.class {
    attr: value;
}
```
클래스 선택자를 사용해 만든 스타일은 클래스 스타일이라고 부른다. 이미 만들어 둔 클래스 스타일을 적용할 때는 `class="클래스 명"`처럼
class 속성을 사용해 지정하면 된다. 클래스 속성은 여러 곳에 동시에 적용이 가능하다.

### 4. id 선택자
<b>id 선택자</b>도 클래스 선택자와 마찬가지로 웹 문서의 특정 부분을 선택해 스타일을 지정할 때 사용한다. 다만 마침표(.) 대신
<b>샾(#)</b>을 사용한다는 점을 제외하면 클래스 선택자와 같다. 웹 요소에 적용시키리면 <code>id="id 명"</code>처럼 id 속성을
사용하면 된다.

하지만 클래스 선택자와 id 선택자의 가장 큰 차이는 <b>클래스 선택자는 앞서 말한 것처럼 여러 번 사용이 가능하나 id 선택자는 문서에서 한 번만
적용이 가능하다</b>는 점이다. id 선택자의 이런 특성때문에 주로 문서의 레이아웃과 관련된 스타일을 지정하거나 웹 요소에 자바스크립트를 사용해 요소를
구분할 때 사용한다.

```HTML
<style>
    #example {
        height: 600px;
        weight: 300px;
        border: 2px solid black;
    }
</style>

<div id="example">
    <!-- 중간 내용 생략 -->
</div>
```

### 5. 같은 스타일 규칙을 사용하는 요소를 묶어서 표현
선택자를 이용해 스타일을 정의하다보면 여러 선택자에 같은 스타일을 적용하고 싶어질 때가 많다.  
이럴 때는 <b>쉼표(,)</b>로 구분해 여러 선택자를 나열한 후 스타일 규칙을 한 번만 정의하면 된다.

```CSS
a,b {
    font: border;
}
```

- <b>선택자(Selector)</b>: 웹 문서의 특정 요소(Element)를 지정하여 스타일을 적용하기 위한 기준.
- <b>전체 선택자 (`*`)</b>: 문서 내의 **모든 요소**에 공통적인 스타일(글꼴 등)을 적용할 때 사용.
- **타입 선택자**: 특정 **태그**를 사용하는 모든 요소에 스타일을 일괄적으로 적용. (예: `span { ... }`)
- <b>클래스 선택자 (`.`)</b>: 마침표를 사용하여 정의하며, 문서 내에서 **여러 번 반복**해서 적용하고 싶은 부분에 사용.
- <b>ID 선택자 (`#`)</b>: 샵 기호를 사용하여 정의하며, 문서 내에서 **단 한 번만** 등장하는 고유한 요소(레이아웃 등)에 사용.
- <b>선택자 그룹화 (`,`)</b>: 쉼표로 여러 선택자를 나열하여 동일한 스타일 규칙을 한 번에 정의할 수 있음.
